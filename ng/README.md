## 通过组件的脚手架生成模板

### install yeoman
```
npm install -g yo
```

### install generator-hfe
```
npm install -g generator-hfe
```

### useage
```
yo hfe:ng
```

按照提示输入你的组件信息：

## 组件开发

通过
```
gulp server 
```
启动组件预览服务

在浏览其中输入：http://localhost:3001/examples/simple.html

then

enjoy your develop

## 单元测试
```
gulp test
```

## 发布
```
gulp publish
```
